# Student Management

### Overview
This is our final project in CS162.

### Requirement
- ncurses library
- g++ compiler

### Installation:
- Clone project from github
```
    git clone https://github.com/khoa3101/Student-Management.git
```
- Build:
```
    make all
```
- Run:
```
    make run    
```
